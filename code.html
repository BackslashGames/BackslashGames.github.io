Minecraft Servers

After playing minecraft for a long time We have decided to create a list of our favourite servers.

We tried to pick servers that are less heard of at the moment, but they still have had a lot of time and effort put into them.

Olympia RPG BETA - MC 1.8.8

Olympia RPG is still in BETA, but that doesn't stop it from giving you a great minecraft RPG experience. With custom mobs, plugins, a custom class system, a dynamic economy, and a custom built map. This really is a great server. Join today at mc.olympiarpg.com .

Olympia Website

Grumpycraft - Multiple Servers

If you like modded minecraft, this is the server for you! It has multiple servers for different modpacks from both the ATLauncher and FTB. What really makes this server great is the community. We played on this server for months and when I walked into this guy's fire jets, everyone stepped in and helped me get my stuff back. (thx bigbad108). Grumpycraft
