Spigot / Bukkit Methods

What Is Spigot?

Spigot is the most widely used server-side modded minecraft software

in the world. It is a continuation of the bukkit API (discontinued in 2010).

It efficiently reduces lag on servers and works well on small servers too.

On this page you can find methods that we have used or created for our own

plugins. You can get spigot at getspigot.org.


To start creating a spigot plugin: download the latest spigot server from getspigot.org.

Make sure you also have eclipse and the java jdk installed.
